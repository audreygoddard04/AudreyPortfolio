# Audrey Goddard - Portfolio Website

Personal portfolio website showcasing projects, achievements, and interests in genetics, health, and athletics.

## About

This is a React-based portfolio website built with Create React App. The site features:

- **Home**: Introduction and overview
- **Projects**: Showcase of research projects, including RHAMM breast cancer thesis proposal, fitness & health content, athletics achievements, and pathology website
- **Bookshelf**: Reading list and book recommendations

## Technologies Used

- React
- React Router
- CSS3
- React Icons

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd AudreyPortfolio/my-app
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
